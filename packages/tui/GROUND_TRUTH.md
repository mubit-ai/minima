## Implementation

```python
def binary_search(array: list, target) -> int:
    """
    Find the index of a target value in a sorted array using binary search.
    
    Preconditions (caller's responsibility):
    - The input array must be sorted in ascending order.
    - All elements in the array must be comparable with the target using < and ==.
    
    Args:
        array: A sorted list of comparable elements.
        target: The value to search for.
    
    Returns:
        The index of the target if found; -1 if not found.
        When duplicates exist, returns one matching index (not guaranteed to be first or last).
    
    Raises:
        TypeError: If elements are not comparable with the target (violates precondition).
    
    Example:
        >>> binary_search([1, 3, 5, 7, 9], 5)
        2
        >>> binary_search([1, 3, 5, 7, 9], 4)
        -1
    """
    lo = 0
    hi = len(array) - 1
    
    while lo <= hi:
        mid = (lo + hi) // 2
        if array[mid] == target:
            return mid
        elif array[mid] < target:
            lo = mid + 1
        else:
            hi = mid - 1
    
    return -1
```

def test_binary_search():

## Notes on Contract and Preconditions

**Unsorted Input:** If the array is not sorted, the function will not validate this and will produce unpredictable results—it may return an index that does not contain the target, return -1 when the target exists, or return any arbitrary match. This is not a bug; it is a violation of the precondition. The caller is responsible for ensuring the array is sorted before calling this function.

**Type Homogeneity:** All elements must be comparable with the target. If the array contains mixed types or incomparable objects, a `TypeError` will be raised during the comparison step. Again, this is the caller's responsibility.

**Duplicates:** When the target appears multiple times, binary search returns *one* matching index, not necessarily the first or last. If you need the leftmost or rightmost occurrence, implement a separate variant with adjusted boundary-update logic.
